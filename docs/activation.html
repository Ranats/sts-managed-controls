Activation

How the unlock flow works

STS2 Managed Controls starts with a 30-minute local trial. After that, managed write actions and bridge commands are gated until the tool receives a valid activation key for the current install ID.

Buyer flow

  1. Launch the desktop UI for STS2 Managed Controls.
  2. Open License -> License Status... and copy the current install ID.
  3. Open the Lemon Squeezy checkout and complete payment.
  4. After payment processing, receive an activation key by email.
  5. Paste the key into the desktop UI or run python -m sts_bot.cli activate-managed-controls --license-key <KEY>.

What the key does

  • The activation key unlocks unlimited access for the licensed install ID.
  • The key is intended for this local single-player desktop toolkit only.
  • Changing hardware, reinstalling the operating system, or major environment changes can require support help.

Support

If delivery fails or the activation key does not work as expected, contact support@auth.cilabworks.com. Include your purchase email and install ID when possible.
