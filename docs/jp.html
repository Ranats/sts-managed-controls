機能

単体プレイの検証ループを短くするための機能群です。

Live State Probe

OCR のみには依存せず、floor, ascension, HP, block, gold, energy を managed snapshot から確認します。

Runtime Actions

Gold の直接変更、block / energy の維持、bridge 経由の power apply を行えます。

Catalog Search

sts2.dll から取得した cards / powers / relics を検索してから操作できます。

Desktop UI

CLI の反復入力や in-game overlay ではなく、独立したローカルウィンドウで操作します。

Fast Test Loops

combat-start auto powers などを使って、検証・調整・デバッグの試行回数を増やせます。

Experimental Paths

Deck replace, relic obtain, map jump, dynamic-var tuning は搭載済みですが experimental 扱いです。

想定ユーザー

こういう用途向けです

  • 単体プレイの sandbox 利用
  • mod 作者の runtime interaction 検証
  • テスターやバランス調整用途
  • OCR 以外の live state inspection を使いたい人

注意点

事前に理解しておくこと

  • ローカル単体プレイ用途のみ
  • 必要ならセーブデータを事前バックアップ
  • experimental bridge action により状態が崩れる可能性あり
  • ゲームアップデートで reflection-based call が壊れる可能性あり
