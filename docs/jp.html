サブスクリプションですか？

いいえ。現在の販売価格は US$6.99 の買い切りです。

購入すると何が手に入りますか？

STS2 Managed Controls を有効化するライセンスキーが届きます。キーは Install ID を使った手順で有効化します。

Install ID をコピーして購入ページで入力し、届いたキーを貼り付けるだけなので、導入手順自体はシンプルです。
