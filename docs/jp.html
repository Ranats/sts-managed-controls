Capabilities

単体プレイの検証ループに寄せた機能構成です。

Live State Probe

OCR だけに頼らず、floor、ascension、HP、block、gold、energy などを managed snapshot path から取得します。

Runtime Actions

Gold の直接変更、block / energy の維持、power の適用などを bridge-based workflow から実行できます。

Catalog Search

sts2.dll から見つけた cards、powers、relics を検索し、そのまま runtime action に繋げられます。

Desktop UI

繰り返し CLI を打たずに、専用のローカル window から操作できます。in-game overlay ではありません。

Fast Test Loops

Combat-start auto power などを使って、mod validation や balance iteration のループを短くできます。

Experimental Paths

Deck replacement、relic obtain、map jump、dynamic-var tuning は experimental 扱いです。

Built For

このツールが向いている用途

  • Single-player sandbox users
  • Runtime interaction を確認したい mod authors
  • テスターや balance iteration 用途
  • OCR-only では足りない live state inspection

Limits

前提と注意点

  • ローカル single-player 用です
  • 必要ならセーブのバックアップを取ってください
  • Experimental bridge actions は unusual state を残すことがあります
  • ゲーム更新で reflection-based bridge calls が壊れることがあります
