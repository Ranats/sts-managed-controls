## STS Managed Controls 0.1.1

This release shifts the public focus of the project to `STS Managed Controls`.

Included in this release:

- refreshed README and public positioning around `STS Managed Controls`
- managed gold write support
- local 30-minute trial and unlock flow for managed-control features
- updated GUI and release media
- improved managed-controls docs and public release docs
- bridge and managed-control workflow updates for powers, cards, relics, and related runtime actions

Public assets in this release:

- `sts-managed-controls-0.1.1.zip`
- `docs/release/gui.png`
- demo videos under `docs/release/`

Notes:

- `STS Managed Controls` is the intended public product surface
- autoplay and automation work remains experimental and is not the main release focus
- the current license gate is a soft local gate, not hardened DRM
