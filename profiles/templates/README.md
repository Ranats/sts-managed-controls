# Template Calibration

Put cropped template images for screen anchors and clickable elements in this folder.

Suggested files:

- `map_anchor.png`
- `reward_anchor.png`
- `battle_anchor.png`
- `rest_anchor.png`
- `shop_anchor.png`
- `game_over_anchor.png`
- `card_slot_1.png`
- `card_slot_2.png`

Template matching works best when:

- the game runs at a fixed resolution
- the window is not scaled by Windows DPI
- crops are tight and distinctive
- matching regions are small
